- Abra o terminal no linux.
- Digite os seguintes comandos:
    javacc AnalisadorLexico.jj
    javac *.java
    java AnalisadorLexico
- Execute os comandos bc que quiser ou utilize os arquivos de entrada na pasta bc
    java AnalisadorLexico < bc/(FileName).bc

ALTERNATIVA JAR

- Execute o comando:
    java -jar AnalisadorLexico.jar < bc/(FileName).bc 
    ou 
    java -jar AnalisadorLexico.jar        // e teste =D
